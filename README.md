# AI-Based Smart Traffic Management System

## Project Overview
This project is an AI and Machine Learning based Smart Traffic Management System developed using YOLO and OpenCV.

The system detects vehicles from traffic videos, analyzes traffic density, and recommends smart traffic signal timing.

---

## Features
- Vehicle Detection using YOLO
- Traffic Density Analysis
- Vehicle Counting
- Smart Signal Recommendation
- Traffic Data Visualization
- CSV Dataset Generation

---

## Technologies Used
- Python
- OpenCV
- YOLOv8
- Pandas
- Matplotlib

---

## Project Workflow
Traffic Video → Vehicle Detection → Vehicle Counting → Density Analysis → Signal Recommendation
